# 光学涡旋参数识别系统 (OV-PRS)

**[English](README.md) | [Русский](README_ru.md) | [日本語](README_ja.md)**

光学涡旋参数识别系统 (OV-PRS) 专为解决根据强度图像识别拉盖尔-高斯光学涡旋参数的专门任务而设计。该程序使用机器学习方法分析从实验装置相机获取的输入数据，并自动确定LG模参数，包括径向指数和方位角指数。基于识别的参数，还可以计算光学涡旋的拓扑荷。

该实现旨在应对真实光学系统中不可避免出现的湍流畸变条件下工作。所使用的神经网络模型在模拟不同湍流水平的数据上训练，即使强度轮廓出现显著畸变，也能确保稳健的参数识别。

该软件是为特定研究任务创建的，但解决方案架构允许其适应其他类型的光学模式、不同的光束参数或不同的实验条件。如有必要，系统可根据用户要求进行修改：更改识别参数集、扩展模型、集成新相机或数据格式，并针对特定光传播条件进行训练。

## 功能特性

- **参数识别**: 确定径向指数 (n)、方位角指数 (m) 和拓扑荷 (TC)
- **工作模式**:
  - 从网络摄像头实时捕获图像
  - 从磁盘加载单个图像
  - 批量处理多个图像
- **导出结果**: 将识别结果保存为CSV格式
- **Web界面**: 通过浏览器在本地主机上工作
- **多语言支持**: 英语、俄语、中文、日语

## 系统要求

- Python 3.8+
- PyTorch 2.0+
- OpenCV 4.8+
- Flask 2.3+
- NumPy, Pillow, torchvision

**注意**: 由于应用程序专门用于光学涡旋识别，因此对神经网络模型有特定要求。

1. 克隆仓库:

```bash
git clone https://github.com/Anxiiie/Optical-Vortex-Parameter-Recognition-System-OV-PRS.git
cd optical-vortex-recognizer
```

2. 安装依赖项:
```bash
pip install -r requirements.txt
```

## 使用方法

### 启动应用程序
```bash
python main.py
```

启动后，打开浏览器并导航至: http://127.0.0.1:5000

### 工作模式

1. **加载模型**:
   - 点击"加载模型"并选择预训练模型文件(.pth或.pt)
   - 模型将被加载并准备就绪

2. **摄像头操作**:
   - 点击"连接摄像头"
   - 应用程序将开始从网络摄像头实时捕获图像
   - 参数识别自动进行
   - 结果显示在右侧面板
   - 点击"停止摄像头"完成

3. **图像加载**:
   - 点击"加载图像"选择单个文件
   - 图像将自动处理
   - 识别结果将在加载后立即显示

4. **批量处理**:
   - 点击"批量加载"选择多个图像
   - 所有图像将按顺序处理
   - 结果将添加到历史记录表

5. **查看和导出结果**:
   - 点击"显示表格"查看所有结果
   - 点击"导出CSV"将结果保存到文件
   - 点击"清除结果"删除所有数据

## 神经网络模型

### 模型要求
- 格式: PyTorch (.pth或.pt)
- 输出: 两个张量的元组(n和m的预测)
- 输入: RGB图像，227x227像素

### 参数范围
- **径向指数 (n)**: 7个类别(从1到7)
- **方位角指数 (m)**: 7个类别(从2到8)
- **拓扑荷 (TC)**: 计算为TC = m - n(始终>= 1)

## 模型准备

应用程序支持以state_dict格式(权重字典)加载预训练的PyTorch模型。

### 模型保存格式

仅保存模型权重(PyTorch开发者推荐):

```python
import torch

# 保存state_dict
torch.save(model.state_dict(), 'model_state_dict.pth')

# 或使用包装器
torch.save({'model_state_dict': model.state_dict()}, 'model_state_dict.pth')
```

应用程序使用AlexNetLG架构加载state_dict。请确保您的模型架构与应用程序的AlexNetLG架构匹配。

## 结果格式

识别结果以CSV格式保存，包含以下列:
- `filename`: 图像文件名
- `n`: 径向指数
- `m`: 方位角指数
- `TC`: 拓扑荷
